# WeChat Article Studio / 公众号顶尖成稿工作流

把一份粗糙草稿，打磨成可以直接发布的公众号文章。  
Turn a rough draft into a publish-ready WeChat article with sharper structure, premium typesetting, and a cohesive visual system.

This repository is a Codex skill bundle for high-end article production. It is built for founders, editors, content teams, and anyone who wants top-tier article writing without the usual noisy, generic template feel.

## What this repo does / 这个仓库做什么

- Produces one primary deliverable: a Word `.docx` ready to open, copy, and paste into the WeChat editor.
- Adds editorial restructuring, title and summary options, visual planning, and a clean release note.
- Includes a paste-ready HTML backup and a disciplined image workflow for figures, covers, and supporting visuals.

## Why it feels premium / 为什么它更高级

- Short, controlled paragraphs that read like edited magazine copy.
- Strong openings, clean section logic, and bold beats that are worth screenshotting.
- A unified visual language for infographics, cover images, and article support art.
- Word-first composition, so the article already looks finished before it reaches the editor.

## Workflow / 工作流

1. Read the style rules and layout guides.
2. Extract facts, figures, and tone from the draft.
3. Rewrite the article with a stronger opening, better pacing, and cleaner section structure.
4. Plan visuals: cover, infographics, and supporting imagery.
5. Render or source assets and embed them into the Word file.
6. Export the HTML backup.
7. Run layout and image QA.
8. Package everything with release notes.

## Repository structure / 仓库结构

- `SKILL.md`: the core operating instructions.
- `references/editorial.md`: voice, structure, title, and summary rules.
- `references/layout-wechat.md`: Word layout, HTML backup, QA, and release notes.
- `references/design-system.md`: infographic system and rendering rules.
- `references/images-sourcing.md`: commercial image sourcing and verification.
- `scripts/`: helper scripts for document assembly, rendering, font setup, and image embedding.

## Philosophy / 方法论

- One article, one point of view.
- One visual system, one voice.
- One main deliverable, a finished `.docx`.
- Clarity first, ornament second, polish without noise.

## Notes / 说明

- This repo is optimized for Chinese-language WeChat publishing, but the introduction and key docs are bilingual.
- It is a skill package, not a web app.
- If you want top-tier article writing with a more editorial feel, this repo is designed for that workflow.
